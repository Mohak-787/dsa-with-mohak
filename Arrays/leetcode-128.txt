/**
* - Solution in Python3
*/
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        numSet = set(nums)
        longest = 0

        for n in numSet:
            if n - 1 not in numSet:
                length = 1

                while n + length in numSet:
                    length += 1

                longest = max(longest, length)

        return longest

/**
* - Step-by-Step Algorithm
*/
Step 1: Convert the input list nums into a set called numSet. This allows for O(1) time complexity lookups and removes duplicates.
Step 2: Initialize a variable longest to 0 to keep track of the maximum sequence length found.
Step 3: Iterate through each number n in numSet.
Step 4: Identify the Start of a Sequence: Check if n - 1 is in numSet. If it is not present, it means n is the first number of a potential consecutive sequence.
Step 5: If n is a starting number, initialize a local variable length to 1.
Step 6: Count the Sequence: Use a while loop to check if the next consecutive numbers (n + 1, n + 2, etc.) exist in numSet. Increment length for every consecutive number found.
Step 7: Update longest by comparing it with the current length using the max() function.
Step 8: After checking all numbers in the set, return the value of longest.
