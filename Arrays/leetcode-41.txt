/**
* - Solution 1 in Python3
*/
class Solution:
    def firstMissingPositive(self, nums):
        nums_set = set()

        for n in nums:
            if n > 0:
                nums_set.add(n)

        i = 1
        while True:
            if i not in nums_set:
                return i
            i += 1

* - Solution 2 in Python3
*/
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)

        for i in range(n):
            if nums[i] <= 0 or nums[i] > n:
                nums[i] = n + 1
        
        for num in nums:
            index = abs(num) - 1
            if index < n:
                nums[index] = -abs(nums[index])
        
        for i in range(n):
            if nums[i] > 0:
                return i + 1
        
        return n + 1

/**
* - Step-by-Step Algorithm for Solution 1
*/
Step 1: Initialize an empty set called nums_set to store unique positive integers.
Step 2: Iterate through each number n in the input list nums.
Step 3: Filter the data: If n is greater than 0, add it to nums_set. (Negative numbers and zero are ignored as they aren't positive integers).
Step 4: Initialize a counter variable i to 1, which is the smallest possible positive integer.
Step 5: Start an infinite while loop to check for the presence of consecutive integers.
Step 6: In each iteration, check if the current value of i exists in nums_set.
Step 7: If i is not found in the set, return i immediately as it is the first missing positive integer.
Step 8: If i is found, increment i by 1 and repeat the check.

/**
* - Step-by-Step Algorithm for Solution 2
*/
Step 1: Determine the length of the input list and store it in a variable n.
Step 2: Clean the Data: Iterate through the array and check if each number is a "valid" candidate (between $1$ and $n$). If a number is less than or equal to $0$ or greater than n, replace it with a placeholder value like n + 1.
Step 3: Mark Presence: Iterate through the array again to treat each number's value as an index.
Step 4: For each number, calculate its absolute value and subtract 1 to get the corresponding index.
Step 5: If this index is within the array bounds (less than n), go to that position (nums[index]) and make the value there negative using -abs(). This acts as a "flag" indicating that the number index + 1 exists in the original list.
Step 6: Scan for the Gap: Iterate through the modified array one last time from index $0$ to $n-1$.
Step 7: Check each value; the first index i that contains a positive number indicates that the integer i + 1 was never marked as present. Return i + 1 immediately.
Step 8: Default Case: If the entire loop finishes and all numbers are negative, it means every integer from $1$ to $n$ was present. Return n + 1.
